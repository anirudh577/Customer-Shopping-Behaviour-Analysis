# 🛒 Customer Shopping Behavior Analysis

📊 A complete **Data Analytics Project** focused on understanding customer shopping behavior using **Python, SQL, PostgreSQL, and Power BI**.
This project covers **data cleaning, EDA, SQL analysis, dashboard creation, reporting, and business recommendations** to uncover valuable customer and sales insights.

---

# 📌 Overview

This project analyzes **3,900 retail transactions** to identify:

* 🧑 Customer purchasing patterns
* 💰 Revenue trends
* 🛍️ Product preferences
* 🚚 Shipping behavior
* ⭐ Review patterns
* 📈 Subscription insights

The workflow includes:

✅ Data Cleaning in Python
✅ SQL-based Business Analysis
✅ Interactive Power BI Dashboard
✅ Professional Reporting & PPT Creation


# 📂 Dataset Information

The dataset contains customer shopping transaction records with features like:

* 👤 Customer demographics
* 🛒 Product categories
* 💳 Purchase details
* 🎟️ Discounts & promo usage
* 🚚 Shipping type
* ⭐ Review ratings
* 🔁 Purchase frequency

### 📊 Dataset Size

* Rows: **3,900**
* Columns: **18**
